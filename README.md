# SpringBoot-02
Sample Spring Boot Rest API with embedded database to execute CRUD operations.
<h5> <i>
Prerequisites : Java 8 </br> 
IDE: Spring STS , Postman client </br> 
Build tool: Maven </br>
Database: Apache Derby (embedded) </br>
